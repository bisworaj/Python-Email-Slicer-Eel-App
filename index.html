Instructions

This is a very simple Email Slicer. This performs a very simple task. We take an Email address from the user and we slice the email address to get the username and domain name.

Then we output the username and domain name back to the user.

The first input field takes an email as input from user and then we run a Python string procesing code to find the the '@' symbol which separate the username and domain name.

Then we slice the email address in 2 parts which is printed inside the output filed.


A sample output will look like this

Email address : email@example.com

Username : email

Domain name : example.com

about the app

This 'Python Email Slicer' app is a simple Interactive Browser bassed app powerd by Python.

in this app we are implimenting Python Eel library to create UI for Python program using HTML and CSS. the Eel allows us to create web Application where the progrm logic runs behind in python. The data/arguments pass on to the python from javascript through the browser. the javascript program basically connects the UI to the python progarm using functions which are trigerred by the events on the browser.

  1. First the user email is recived through the input field and goes to javascript
  2. the email is verifyed by the javascript by some validation functions
  3. then javascript passes the email as an argument to the python script.
  4. The string processing function is executed in the python script
  5. the eel comunicates with javascript and retuns the result using a callback function
  6. the desired output is then printed in the DOM.
